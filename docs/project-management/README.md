## Project Overview

The Research Notebook is an electronic lab notebook application built with:
- **Frontend**: React + TypeScript + Vite
- **Backend**: Node.js + Express + Prisma
- **Desktop**: Electron
- **Database**: SQLite/PostgreSQL
